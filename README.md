# covelab-foundation

Universal Foundation for AI-assisted project work.

A reusable framework for solo founders, consultants, and AI builders
working with Claude Code — without needing to be a programmer.

## What is this?

This repository contains the Universal Fundament framework layer:
- ADR system (Architecture Decision Records)
- Best-practice research snapshots
- Framework documentation (FAHRPLAN, STARTPROMPT, SKILL_INDEX)
